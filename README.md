## Overview
This project is a **modern web application** built as a **monorepo using Turborepo** and **Bun**.  
It is designed to handle multiple packages efficiently, supports fast development workflows, and includes a scalable structure for future growth.

Key highlights:
- Monorepo architecture for multiple packages
- Efficient build & caching using Turborepo
- Fast runtime and package management with Bun
- Environment-specific configuration and easy deployment
